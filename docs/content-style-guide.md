# Puzzlepia Content Style Guide

Puzzlepia content should feel friendly, global, premium casual, and easy to understand. The site is an early official puzzle hub from BornstarSoft, so copy should be confident but modest.

## Naming

Use these names consistently:

- Puzzlepia
- BornstarSoft
- KPOP Word Search
- Blockzzle
- Ringzzle
- Mapdoku
- Color Rings Mania
- Daily Puzzle
- Daily Challenge
- Terms of Use
- Privacy Policy

## Tone

- Friendly and clear.
- Simple global English.
- Helpful without sounding stiff or overly educational.
- Calm, bright, and lightly premium.
- Honest about the current early-stage site.
- Avoid hype that makes Puzzlepia sound like a large game portal.

## Blog Writing Style

- Begin with the reader's practical puzzle interest.
- Use short paragraphs and clear headings.
- Keep each post focused on one topic.
- Mention Puzzlepia naturally, usually once near the beginning and once near the end.
- Link to related Daily Puzzle and Games pages when useful.
- Do not use fake statistics or forced keyword repetition.

## Daily Puzzle Writing Style

- Keep the prompt short and playable in the reader's mind.
- Include a hint and answer section.
- Make the difficulty feel approachable.
- Link to one playable Daily Puzzle page and one related game page.
- Do not imply dynamic daily generation unless it exists.
- Avoid making static challenge pages sound like full games.

## Game Page Writing Style

- Describe the game type, mood, and player appeal.
- Keep status labels accurate.
- Use confirmed store links only.
- If a store link is unknown, say "Store link coming soon" or "Follow Puzzlepia for updates."
- Connect each game to a related Daily Puzzle where possible.
- Avoid inventing features, release dates, screenshots, or platform availability.

## Internal Linking Rules

- Link Blog posts to `/daily/`, `/games/`, and specific relevant pages.
- Link Daily Challenge pages to one playable daily puzzle and one related game page.
- Link game pages back to related daily puzzles.
- Keep links natural in the sentence.
- Do not overload a paragraph with links.

## SEO Titles and Descriptions

- Titles should be clear, specific, and readable.
- Descriptions should summarize the page in one concise sentence.
- Use unique descriptions for every page.
- Avoid clickbait and exaggerated claims.
- Keep title and description aligned with the actual content.

## Phrases to Avoid

- "Huge game portal"
- "Massive game library"
- "Thousands of puzzles"
- "New generated puzzle every day" unless it is technically true
- "Best in the world"
- "Guaranteed to improve your brain"
- "Scientifically proven" unless supported by a real source
- "Compete on rankings" unless rankings exist
- "Create an account" unless accounts exist
- "Buy" or "payment" language for Puzzlepia website features
