# Puzzlepia Blockzzle Cross-Promotion Note

Puzzlepia links to Blockzzle as an official external BornstarSoft puzzle game.

## Links Added

- Game site: `https://blockzzle.com/`
- Play: `https://blockzzle.com/play/`
- Leaderboard: `https://blockzzle.com/leaderboard/`
- Block puzzle overview: `https://blockzzle.com/block-puzzle/`

## Puzzlepia Placement

- `/games/blockzzle/` describes Blockzzle as a free browser-playable block puzzle game.
- Shared game cards can show `Play Blockzzle`, `Leaderboard`, and the internal Puzzlepia game detail link when a game page provides those URLs.
- The existing homepage Featured Games and `/games/` list include Blockzzle through the normal Hugo games collection.
- Related Block Puzzle blog posts and Daily Challenge pages already connect readers toward `/daily/block-puzzle/`, `/games/blockzzle/`, and the official Blockzzle play page.
- Selected high-relevance Block Puzzle pages may include `https://blockzzle.com/leaderboard/` as a secondary official leaderboard link after the primary play link.

## Distinction From Ringzzle

- Blockzzle is positioned as a block puzzle game about placing block pieces, clearing lines, and chasing a best score.
- Ringzzle remains positioned as a separate color rings puzzle game about placing rings, clearing matching color lines, and Color Burst moments.
- The two games may appear together in Puzzlepia game lists, but they should not be described as the same game.

## Safety Notes

- No fake rankings, fake popularity, fake reviews, fake download numbers, fake traffic claims, or fake player counts were added.
- No reward, monetization, or ranked-competition claims were added.
- No gameplay, backend, analytics, ads, login, email capture, IAP, monetization, or Daily Challenge features were added.
- No Blockzzle or Ringzzle repository files were modified.
- Puzzlepia remains the portal; Blockzzle remains an external official game link.
