## Future HTML5 Puzzle Improvements

- Add clearer win-state animations without sound.
- Add better keyboard focus states for playable puzzles.
- Add a small hint button for each starter puzzle.
- Add deterministic puzzle variants by page, not by user data.
- Improve touch feedback on small screens.
- Add simple accessibility labels for dynamic status updates.
- Keep reset behavior reliable before adding new mechanics.

## Games Hub Improvement Ideas

- Add confirmed store links as they become available.
- Add screenshots only when official assets are ready.
- Add a "related daily puzzle" card to each game page.
- Add lightweight update notes for games with changing availability.
- Keep status labels accurate: Available, Coming soon, In development, or Open beta.
- Avoid inventing dates, store URLs, or feature claims.
