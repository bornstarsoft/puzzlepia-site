# Ringzzle Cross-Promotion Note

Puzzlepia links to Ringzzle as an official external BornstarSoft puzzle game.

- Game site: `https://ringzzle.com/`
- Play: `https://ringzzle.com/play/`
- Leaderboard: `https://ringzzle.com/leaderboard/`
- How to Play: `https://ringzzle.com/how-to-play/`

## Puzzlepia Placement

- `/games/ringzzle/` describes Ringzzle as a free browser-playable color rings puzzle game.
- Shared game cards can show `Play Ringzzle`, `Leaderboard`, and the internal Puzzlepia game detail link when a game page provides those URLs.
- The existing homepage Featured Games and `/games/` list include Ringzzle through the normal Hugo games collection.

## Safety Notes

- No fake rankings, fake popularity, fake reviews, or fake player counts were added.
- No gameplay, backend, analytics, ads, login, email capture, IAP, monetization, or Daily Challenge features were added.
- No Ringzzle repository files were modified.
- Puzzlepia remains the portal; Ringzzle remains an external official game link.
