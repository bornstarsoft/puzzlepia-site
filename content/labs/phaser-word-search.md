This is an experimental Phaser-powered Word Search prototype for Puzzlepia Labs.

It is available here for lab testing only. Puzzlepia's stable Daily Word Search remains available as the main Daily Puzzle version.

## What this lab tests

- Smoother touch selection on mobile.
- Clearer word highlighting and selection feedback.
- A self-contained static game embed.
- A safe comparison against the current Vanilla JavaScript Daily Word Search experience.
