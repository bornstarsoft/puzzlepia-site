This challenge helps you think about color and ring size together.

## Puzzle prompt

You have three pink rings: large, medium, and small. Place them so each ring goes into a cell that can hold its size.

Try to keep the board readable and avoid duplicating the same size in one cell.

## Hint

A cell can hold different sizes, but it should not hold two rings of the same size.

## Answer

One simple solution is to place each pink ring in a different cell. This keeps every cell flexible and makes the color pattern easy to see.
