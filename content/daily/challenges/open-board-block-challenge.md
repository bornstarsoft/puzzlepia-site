This challenge is about protecting open space.

## Puzzle prompt

You have three pieces: a 3-block line, a 2x2 square, and a single block.

Which piece should you usually place first if the board has one large open area?

## Hint

Larger pieces need more room. Smaller pieces can usually fit later.

## Answer

Place the 2x2 square or the 3-block line before the single block. The single block is flexible, so it is safer to save it until the larger shapes have a clean place.
