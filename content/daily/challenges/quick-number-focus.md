This quick challenge warms up number order.

## Puzzle prompt

Tap these numbers in ascending order:

`4  1  3  2`

Write the correct order before you play.

## Hint

Ascending order means smallest to largest.

## Answer

The correct order is `1, 2, 3, 4`.
