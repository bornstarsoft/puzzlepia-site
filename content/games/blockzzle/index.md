Blockzzle is a free block puzzle game from BornstarSoft that you can play online in your browser. Place block pieces on the board, clear lines, and keep building your score in a quick no-install puzzle session.

Start playing on the official Blockzzle site: [Play Blockzzle](https://blockzzle.com/play/).

Want the overview first? [Learn more about this block puzzle game](https://blockzzle.com/block-puzzle/).

Blockzzle also has an [anonymous leaderboard](https://blockzzle.com/leaderboard/) for submitted scores. It is nickname-only and does not require login or email.

You can also visit the [Blockzzle game site](https://blockzzle.com/) for the main game page.

If you are arriving from Puzzlepia's Daily Puzzles, start with the lightweight [Daily Block Puzzle](/daily/block-puzzle/) to practice the basic placement flow.
